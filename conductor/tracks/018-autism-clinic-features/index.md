# Track 018: Autism Clinic Integration

## Overview
This track implements features specialized for an Autism clinic, focusing on stress detection, event tagging, and clinician-facing data summaries.

## Goals
-   [ ] Implement real-time stress detection logic (HR/HRV delta).
-   [ ] Add a tagging system to alerts (Sensory Overload, Transition, etc.).
-   [ ] Create a baseline calibration tool for personalized stress thresholds.
-   [ ] Implement a caregiver/clinician notification bridge.

## Next Steps
1.  Define the Stress/Overstimulation evaluation algorithm.
2.  Update the `Alert` data model to include tags.
3.  Implement the tagging UI in the mobile app.
