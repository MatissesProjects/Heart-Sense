# Implementation Plan: Track 024 - Active Biofeedback

## Overview
Introduce interactive calming tools on the watch that respond to physiological stress.

## Sub-tasks
1. **Biofeedback UI:** Create a Wear Compose screen for breathing guidance.
2. **Haptic Engine:** Implement a utility to trigger rhythmic vibration patterns.
3. **Stress-to-Action Bridge:** Update `PassiveMonitoringService` to show a prompt/notification for biofeedback when stress is high.
4. **Post-Exercise Analysis:** Compare HR/HRV before and after the exercise to verify effectiveness.
