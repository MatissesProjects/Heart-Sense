# Heart-Sense: Workflow

## Development Process
1.  **Branching:** Use descriptive branch names for each track (e.g., `track/001-setup`).
2.  **Implementation:** Follow the track-specific plan.
3.  **Testing:** Ensure unit tests and integration tests (where applicable) are added.
4.  **Verification:** Verify on physical devices (Pixel 9 & Pixel Watch 3) if possible, otherwise use emulators.
5.  **Commit:** Commit logical chunks with clear messages.
